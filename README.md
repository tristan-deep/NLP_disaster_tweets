## Achieved results

| Model              | Train acc | Val acc | Train loss | Val loss |
|--------------------|-----------|---------|------------|----------|
| LSTM               | 0.95      | 0.72    | 0.13       | 1.91     |
| Bidirectional LSTM | 0.96      | 0.72    | 0.10       | 2.57     |

| CNN + LSTM         | 0.84      | 0.77    | 0.38       | 0.58     |
